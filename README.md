# Maestría en Ciencia de Datos - Actividades de Big Data

Este repositorio está destinado a la entrega y organización de las actividades desarrolladas en el espacio académico de **Big Data** del programa de **Maestría en Ciencia de Datos**.

Aquí se encuentran los archivos (notebooks, python, sql, etc.), conjuntos de datos y documentación técnica relacionada con el procesamiento y análisis de datos masivos.

## Actividades del Curso

### 1. Actividad de Reconocimiento: Databricks
En esta actividad se realiza una exploración inicial del entorno de Databricks, incluyendo la carga de datos, almacenamiento en formato Delta y consultas básicas utilizando SQL y PySpark.
